Type scale
Display
As the largest text on the screen, display styles are reserved for short, important text or numerals. They work best on large screens.
Headline
Headline styles are smaller than display styles. They're best-suited for short, high-emphasis text on smaller screens. These styles can be good for marking primary passages of text or important regions of content.
Title
Titles are smaller than headline styles and should be used for shorter, medium-emphasis text.
Label
Label styles are smaller, utilitarian styles, used for areas of the UI such as text inside of components or very small supporting text in the content body, like captions.
Body
Body styles are used for longer passages of text. Use typefaces intended for body styles. This helps ensure readability at smaller sizes and in longer passages.
This type scale is based on Material’s default type scale. To learn more about the latest type guidelines, please visit https://m3.material.io/styles/typography/overview
